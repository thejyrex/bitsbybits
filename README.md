# Bits by Bits

A habit tracking application that helps you build better habits and achieve your goals through progressive tracking and smart notifications.

<img src="https://github.com/user-attachments/assets/b4ced10e-ab88-4c08-9a1f-7cf1002ef6da" alt="BBB GIF" height="720">

## Features

### Core Functionality
- **Habit Management**: Create, edit, and delete habits with customizable schedules
- **Goal Organization**: Create goals you want to achieve then assign habits to it
- **Progress Tracking**: Visual progress tracking with charts and statistics
- **Smart Notifications**: Custom notification times to remind you to do your scheduled habit

### Scheduling Options
- **Daily Habits**: Track everyday activities
- **Weekly Habits**: Set specific days of the week
- **Monthly Habits**: Monthly recurring habits
- **Custom Schedules**: Flexible repeat patterns for unique needs

### Advanced Features
- **Statistics & Analytics**: Detailed progress charts and habit completion rates
- **Backup & Restore**: Export/import utilizing a separate portable backup file
- **Internationalization**: Currently supporting EN and ID locale

### Data Management
- **Local Storage**: All your habits and goals and its progression are stored locally in your device
- **File Operations**: Import/export capabilities with secure file handling

## Installation

1. Go to the release page (https://github.com/thejyrex/bitsbybits/releases) 
2. Find which app version you want to try then expand the assets dropdown
3. Download and install the .apk on your device
